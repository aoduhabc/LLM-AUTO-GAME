## 🎮 项目概览

**《乡野絮语》**是一款基于 LLM 驱动的乡村探索互动小游戏。  
玩家在轻量 2D 地图中自由漫游，进入不同区域会触发对应 NPC，与其进行多轮对话。  
NPC 回复由大模型动态生成，并结合角色设定、场景信息与对话记忆，呈现更自然、连续的互动体验。

---

## ✨ 创新性亮点

### 🧑‍🌾 1) AI 角色不是“固定脚本”，而是“可持续关系”
- NPC 具有角色卡（姓名、职业、性格、背景）与记忆摘要。
- 同一 NPC 会根据历史对话“认出玩家”，让聊天从“问答”变成“关系发展”。
- 回复不是固定句库，而是结合上下文动态生成。

### 🗺️ 2) 区域探索 × 叙事生成的融合玩法
- 进入村路、麦田、桥边、果园、集市、茶棚等区域时，触发不同氛围事件与人物语境。
- 每一次到访都可能产生新的文本体验与故事碎片，增强“轻剧情 + 轻探索”沉浸感。

- 上手门槛低：进入地图就能探索、遇人、开聊，几乎不用学习成本。
- 内容表达强：同一玩法循环下，文本反馈会因角色与时机变化而保持新鲜感。
- 扩展空间足：可自然接入任务系统、好感度成长、多地图冒险等长期玩法。

## 🧭 可玩性设计

### 🔁 核心循环
1. **移动探索**：在地图中行走，进入不同区域。  
2. **遭遇角色**：区域生成/出现 NPC。  
3. **发起对话**：玩家输入问题或点击推荐话题。  
4. **动态反馈**：NPC 按设定与记忆生成回复。  
5. **积累关系**：聊天次数与历史摘要推进“陌生 → 信任”关系变化。  

### 🎯 为什么“好玩”
- **可重复游玩**：同一地区、同一 NPC 在不同时刻有不同表达。  
- **轻操作重反馈**：操作简单，但文本反馈密度高、情感细腻。  
- **高代入感**：头像 + 场景 + 语言风格统一，构成温暖治愈的乡村气质。

## 🎨 视觉风格

- 主色调以麦黄、橄榄绿、木色为主，营造“柔和、安静、治愈”氛围。
- 玩家与 NPC 采用头像化展示，强化角色识别与对话代入。
- 界面简洁，强调“看得见场景、读得懂情绪、聊得出故事”。

---

## 🕹️ 游玩体验机制（简述）

- **区域即关卡氛围**：进入不同地点会触发差异化氛围、见闻与社交语境，形成“走到哪、玩到哪”的体验节奏。  
- **角色即长期关系**：每位 NPC 都会记住与你的互动轨迹，对话从寒暄逐步演进为关系推进。  
- **对话即主要反馈**：玩家每次提问都能获得情绪化、语境化回应，持续给到“被回应”的满足感。  
- **探索即内容解锁**：区域访问会累积故事碎片、传闻与动态事件，让短时游玩也有可见成长。

## 🆕 最新功能更新

### 🤖 NPC 自治系统（每个 NPC 都有自己的生活节奏）
- 每位 NPC 都有自己的“近况轨迹”，会持续积累个人见闻与状态变化。
- 即使玩家暂时不和他对话，角色也会在村庄中继续活动，世界更有“在运行”的感觉。

### 🧠 行为节奏优化（更稳定、更真实）
- 角色行动不再机械重复，会在“闲逛、走动、传播见闻、制造现场动静”之间自然切换。
- 行动节奏更克制，减少连续刷同类行为，提升观察体验。
- 加入昼夜行为差异：夜晚活动更保守，整体氛围更贴近乡村日常。

### 🌐 跨 NPC 影响链（世界会“连锁反应”）
- 一个 NPC 的举动会影响同区域其他人，村里会出现可感知的“话题传递”。
- 你会看到连锁变化：先是某人起头，随后周围角色陆续出现呼应行为。
- 群体表现更连贯，玩家更容易读懂“这个村子正在发生什么”。

- NPC 面板新增“自治会话”与“近期动作”展示。

- 一键启动更稳定，减少重复启动导致的冲突问题，进入游戏更顺畅。
